# 🚀 Promptrix – AI Prompt Marketplace

Promptrix is a modern AI prompt marketplace built with **Next.js**, **MongoDB**, and **Better Auth**. It allows creators to publish AI prompts, users to discover and purchase premium prompts, and administrators to moderate content through a secure role-based dashboard. The platform provides a seamless experience for prompt sharing, monetization, and community engagement.

## 📖 Project Overview

Promptrix was developed to demonstrate a complete creator marketplace where prompt creators can showcase their work, users can explore and unlock premium AI prompts, and administrators can efficiently manage platform content. The application combines secure authentication, payment integration, analytics, and responsive design into a modern full-stack web application.

## ✨ Core Features

### 🤖 AI Prompt Marketplace

* Browse public AI prompts.
* Search prompts by keywords.
* Filter and sort prompts.
* Pagination for smooth browsing.

### ✍️ Creator Dashboard

* Submit AI prompts.
* Upload prompt thumbnails.
* Manage published prompts.
* Track prompt performance with analytics.

### 🔒 Authentication & Authorization

* Secure authentication using Better Auth.
* Google Sign-In support.
* Role-based access for Users, Creators, and Admins.
* Protected routes for authorized users.

### 💎 Premium Prompt Access

* Purchase premium prompts through Stripe.
* Unlock private prompt content after successful payment.
* Secure premium content management.

### ⭐ Community Features

* Bookmark favorite prompts.
* Submit reviews and ratings.
* Report inappropriate prompts.
* View creator profiles.

### 🛡️ Admin Dashboard

* Approve or reject submitted prompts.
* Manage platform users.
* Moderate prompt content.
* Monitor marketplace activity.

## ⚙️ Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/FBushra-git/your-repository-name.git
```

### 2. Navigate to the project folder

```bash
cd your-repository-name
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env.local` file and add the required variables.

```env
MONGODB_URI=your_mongodb_connection_string
BETTER_AUTH_SECRET=your_secret_key
STRIPE_SECRET_KEY=your_stripe_secret_key
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```

### 5. Start the development server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```
